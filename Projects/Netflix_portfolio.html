Netflix Home Page

Project information

  • Category: Web design
  • Project date: 14 September, 2023
  • Project URL: https://midhunkalarikkal.github.io/Netflixhomepage/Index.html

I created a captivating Netflix-inspired homepage using HTML and CSS, leveraging the power of CSS flex properties for seamless layout. The webpage boasts multiple scrolling sections, each offering unique content and information. At the centerpiece is a prominent movie banner showcasing a featured film alongside its relevant details, creating an engaging focal point for visitors. The navigation bar neatly organizes additional links, enhancing user experience and enabling seamless exploration of the webpage's rich and diverse content.
